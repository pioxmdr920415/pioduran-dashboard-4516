## üêõ Troubleshooting

### Installation Issues

**Problem**: Install button doesn't appear
- **Solution**: 
  - Ensure you're using a supported browser
  - Try accessing via HTTPS (not HTTP)
  - Clear browser cache and reload
  - Check if app is already installed

**Problem**: Installation fails
- **Solution**:
  - Check available disk space
  - Ensure browser is up-to-date
  - Try restarting your browser
  - Check browser settings allow installations

### Performance Issues

**Problem**: App loads slowly
- **Solution**:
  - Clear app cache from settings
  - Check internet connection
  - Update to latest version
  - Restart the app

**Problem**: Offline mode not working
- **Solution**:
  - Ensure you accessed content while online first
  - Check browser storage settings
  - Verify service worker is registered
  - Try reinstalling the app

### Data Issues

**Problem**: Data not syncing
- **Solution**:
  - Check internet connection
  - Manually trigger sync
  - Check if server is accessible
  - Clear cache and re-sync

**Problem**: Old data showing
- **Solution**:
  - Manually trigger sync
  - Clear app cache
  - Check last sync time in settings
